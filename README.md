# CSharp.CustomCache

Implémentation personnalisée d'un provider de cache en C# avec stratégie d'expiration configurable.

## Structure

- `CustomCache/CustomCacheProvider.cs` — Provider de cache avec gestion d'expiration
- `CustomCache/Program.cs` — Exemple d'utilisation
- `UnitTestProject/UnitTest.cs` — Tests unitaires du cache

## Stack

C# / .NET Framework
